Automated Book racking System

System makes the existing library management more easier, where each book is arranged automatically by scanning its QR code by an android app.Books are arranged according to its category or details.Borrowing and returning can be controlled by the admin using a website that is developed.The app has voice recognition feature hence the required book can be obtained just by saying the books name. All the user details such as name,books taken,due date and fine can be viewed at the website.
